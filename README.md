# Neural Data Testing and Functional Specs

This repository is a personal portfolio adaptation of a University of Washington team project on analyzing human and monkey neural recordings from NWB-formatted datasets.

This version focuses on my individual contributions: functional specifications, test design, and reproducibility-oriented project structure.

## Project Overview

The original team project analyzes neural recordings from human and monkey datasets and provides reusable code for loading, formatting, analysis, and visualization.

The project package includes modules for:
- data loading
- data formatting
- analysis
- visualization
- configuration management

## My Contribution

My primary contributions in the original team project were:
- writing functional specifications
- designing and writing tests
- supporting clearer project validation and usability

This portfolio version highlights how I approached:
- translating project requirements into testable behaviors
- defining expected functionality
- improving confidence in correctness and reproducibility

## Repository Structure

- `src/`: source package
- `tests/`: test suite
- `examples/`: example usage
- `docs/`: portfolio-focused explanation of specs and testing decisions
